# Tela Inicial do Twitter — Flutter

Este projeto consiste na recriação da tela inicial do aplicativo Twitter, desenvolvida em Flutter como parte de uma atividade prática focada em construção de interfaces.

O principal objetivo foi reproduzir fielmente o layout original, aplicando conceitos essenciais do Flutter, como:
- Organização de layout com Column, Expanded e Align
- Criação de botões personalizados com InkWell e Material
- Componentização de widgets reutilizáveis
- Alinhamento, espaçamento e hierarquia visual
- Boas práticas de UI com Flutter

A interface foi construída de forma responsiva, priorizando clareza visual, reutilização de código e fidelidade ao design original.

## Tecnologias utilizadas
- Flutter
- Dart
- Material Design

## Elementos da tela
- Botões sociais (Google e Apple)
- Botão principal “Create Account”
- Links de Terms, Privacy Policy e Cookie Use
- Texto de login para usuários existentes

Projeto desenvolvido com foco em aprendizado, prática de UI e evolução no desenvolvimento mobile com Flutter.
